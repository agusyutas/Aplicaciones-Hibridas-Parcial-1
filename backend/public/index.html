API de Fichas Técnicas de Autos

Esta API de Fichas Técnicas de Autos sirve para manejar información de usuarios, autos y marcas. Se hizo con Node.js, Express y MongoDB, y funciona con endpoints que permiten consultar, agregar, actualizar o eliminar información, y todo se maneja en formato JSON.

Con respecto a los usuarios, la API permite registrar nuevos usuarios con sus datos básicos (nombre, email y contraseña), listar a todos los que ya están guardados, consultar un usuario específico a través de su ID, modificar sus datos si fuera necesario o directamente eliminarlos de la base de datos.

En cuanto a los autos, se pueden consultar todos los modelos disponibles y también agregar autos nuevos con sus características completas: marca, modelo, año, tipo de motor, potencia, velocidad máxima y tipo de combustible. Además, es posible buscar un auto en particular por su ID, editar sus datos si hay cambios o eliminarlo si ya no corresponde tenerlo en la base de datos.

Por el lado de las marcas, la API maneja la información de cada fabricante, permitiendo cargar el nombre de la marca junto a su país de origen. Al igual que con los usuarios y autos, se pueden listar todas las marcas disponibles, consultar una marca en específico mediante su ID, modificar sus datos o eliminarla.

Usuarios:

  • GET /api/usuarios
  • POST /api/usuarios 
                                    {
                                 "name": "Agustin",
                                 "email": "agus@gamil.com"
                                }
  • PUT /api/usuarios 
                                    {
                                 "name": "Agustin Y.",
                                 "email": "agusnuevo@gmail.com"
                                }
